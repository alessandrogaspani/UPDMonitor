# Configurazione

## Requisiti

- Windows
- .NET 9
- Visual Studio recente con supporto WPF

## Percorsi applicativi

L'applicazione usa i seguenti percorsi relativi alla cartella eseguibile:

- `Config` per i file di configurazione
- `Logs` per i log

## Configurazione file

Il file di configurazione principale viene letto da:

- `Config\config.json`

## Nota

Se la cartella `Config` o `Logs` non esiste, l'app può crearla quando necessario.
