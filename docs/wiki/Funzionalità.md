# Funzionalità

## Canale inbound

- ascolto su una porta UDP configurata
- ricezione di payload byte[]
- conversione e visualizzazione dei messaggi nell'interfaccia
- conteggio messaggi e statistiche base
- pulizia della lista messaggi
- esportazione CSV

## Canale outbound

- configurazione di porta e indirizzo IP remoto
- invio di messaggi UDP
- collegamento e scollegamento del canale
- supporto all'auto-send

## About

- copia diagnostica negli appunti
- apertura cartella log
- apertura repository GitHub
- visualizzazione informazioni runtime e sistema

## UI

- shell principale con navigazione a regioni Prism
- dialoghi modali per dettagli, about e creator
- finestra principale personalizzata con comandi di title bar
