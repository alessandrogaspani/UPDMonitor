# Esecuzione

## Avvio in locale

1. Aprire la soluzione in Visual Studio
2. Impostare `UDPMonitor` come progetto di avvio
3. Avviare l'applicazione

## Flusso tipico

### Inbound

1. Aprire la schermata Inbound
2. Impostare la porta di ascolto
3. Avviare la ricezione
4. Inviare dati UDP verso quella porta da un client esterno
5. Osservare i messaggi nella lista

### Outbound

1. Aprire la schermata Outbound
2. Impostare porta e IP remoto
3. Connettersi
4. Inserire il messaggio
5. Inviare il payload
