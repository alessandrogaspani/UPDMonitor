# Contributi

## Come contribuire

1. Crea un branch dedicato
2. Implementa la modifica
3. Aggiorna test e documentazione
4. Verifica build e test
5. Apri una pull request

## Suggerimenti

- mantieni le modifiche piccole e mirate
- preferisci lo stile già presente nel repository
- aggiungi test per il comportamento nuovo o modificato
- aggiorna la wiki quando cambia il flusso applicativo
